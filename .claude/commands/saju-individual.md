## 전제 원칙

| 항목 | 기준 |
|---|---|
| **이론 기준** | 이석영 『사주첩경』 우선 |
| **이견 처리** | 현대 명리학 차이는 `> 📌 현대 명리 관점` 블록으로 병기 |
| **데이터 소스** | saju-parse `analyze_saju()` 반환값 그대로 사용 |
| **만세력 변환** | Claude 수행 불가. 사용자가 천간지지 직접 제공 |
| **문체** | 존댓말, 단언 대신 경향성 표현 ("~하는 경향", "~할 수 있음") |
| **출력 형식** | Markdown. 섹션 순서 고정 |
| **주의 문구** | 리포트 하단 필수 삽입: "사주는 경향성이며 운명이 아닙니다" |

## 입력 데이터 명세

saju-parse의 `analyze_saju()` 반환 딕셔너리를 그대로 수신한다.
추가 입력: 이름(선택), 생년월일(참고용, 계산 미사용)

```python
data = {
  # saju-parse와 완전 일치 유지 필수
  'pillars':        [(년간,년지),(월간,월지),(일간,일지),(시간,시지)],
  'day_stem':       str,           # 일간 (예: '壬')
  'gender':         str,           # '남' / '여'
  'element_counts': {'木':int,'火':int,'土':int,'金':int,'水':int},
  'naeum':          [(한자,오행,한글), ...],  # 4개 (인덱스 0=년,1=월,2=일,3=시)
  'stem_sipsin':    [str, ...],    # 4개 천간 십신
  'branch_sipsin':  [str, ...],    # 4개 지지 십신 (본기 기준)
  'jijanggan':      [[str,...], ...],         # 4개 지장간 리스트
  'stages':         [(한자,한글), ...],       # 4개 십이운성
  'relations':      {'충':[],'형':[],'파':[],'해':[],'합':[],'삼합':[]},
  'gongmang_day':   [str, str],    # 일주 기준 공망 지지 2개
  'gongmang_year':  [str, str],    # 년주 기준 공망 지지 2개
  'wollyeong':      str,           # 월령 천간
  'cheonul':        [str, str],    # 천을귀인 지지 2개
  'stem_elements':  [str, ...],    # 천간 오행
  'branch_elements':[str, ...],    # 지지 오행
}
```
