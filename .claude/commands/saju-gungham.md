# 사주 궁합 분석 스킬

## 개요

2인 이상 여러 사람의 사주팔자(년주·월주·일주·시주)를 입력받아 명리학 이론에 따라 궁합을 분석한다.  
단순 궁합 점수를 내는 것이 아니라, **왜 맞고 왜 안 맞는지**를 이론적으로 설명하고 실용적 조언까지 제공하는 것이 목표다.

### 이론적 기반

**이석영의 사주첩경**을 1차 이론 기반으로 삼는다.

- 격국론(格局論)을 사주 해석의 출발점으로 사용
- 용신은 억부·조후·통관·병약·전왕의 5가지 방법론 순서로 적용
- 일주(日柱) 중심의 궁합 판단
- 합충의 길흉은 용신 맥락에서 판단 (합이 무조건 좋은 것이 아님)

현대 명리에서 수정·보완된 해석이 필요한 경우 `(현대 명리)`라고 명시하여 사주첩경 원칙과 구분한다.  
세부 이론은 `references/sajucheomgyeong.md` 참조.

인원 수에 따라 분석 방식이 달라진다:
- **2인**: 상호 분석 (표준 궁합)
- **3인 이상**: 쌍별 분석 + 그룹 전체 역학 분석

---

## 입력 형식

사용자로부터 모든 인원의 사주팔자를 받는다. 형식은 유연하게 수용한다.

**표준 입력 예시 (3인):**
```
A(이름 또는 역할): 甲子 / 丙午 / 壬寅 / 庚申
B(이름 또는 역할): 己卯 / 辛亥 / 乙未 / 丁酉
C(이름 또는 역할): 丁巳 / 壬子 / 甲午 / 癸亥
```

- 년주 / 월주 / 일주 / 시주 순서
- 한자(甲子) 또는 한글(갑자) 모두 수용
- 시주를 모르는 경우 3주만으로 분석 가능 (단, 시주 미상임을 명시)
- 인원에 제한 없음. 단 5인 이상이면 쌍별 분석은 요약 위주로 처리하고 그룹 역학에 집중

입력이 불완전하면 분석 전에 반드시 확인한다.

## 분석 프레임워크

### 1단계: 기본 구조 파악 (사주첩경 원칙)

각 사람의 사주를 먼저 개별 정리한다.

- **격국(格局)**: 월지를 기준으로 격을 확인 (정관격, 식신격, 상관격 등) → `sajucheomgyeong.md` 격국론 참조
- **일간(日干)**: 오행과 음양 확인
- **신강/신약 판단**: 득령·득지·득세 기준
- **용신 추론**: 억부 → 조후 → 통관 → 병약 순으로 적용 → `sajucheomgyeong.md` 용신론 참조

> ⚠️ 용신은 사주 전체 맥락에서 달라진다. "추정 용신"으로 표현하고 단정하지 않는다.

### 2단계: 일간 오행 관계 분석

두 사람의 **일간끼리의 오행 생극 관계**를 분석한다.

| 관계 | 의미 | 해석 |
|------|------|------|
| 상생(相生) | A가 B를 생함, 또는 B가 A를 생함 | 한쪽이 에너지를 공급 → 의존/헌신 관계 가능 |
| 비겁(比劫) | 같은 오행 또는 같은 일간 | 동류 의식 강함, 경쟁 또는 동지 |
| 상극(相剋) | A가 B를 극함 | 통제/억압 구도, 관계 긴장 |
| 상모(相侮) | 극하려는 쪽이 오히려 약함 | 반발·갈등 구조 |

**주의**: 생이 무조건 좋고 극이 무조건 나쁜 게 아니다. 각자의 용신 맥락에서 해석해야 한다.

### 3단계: 용신 교환 관계 분석 (핵심)

궁합의 핵심은 **"상대방이 나의 용신을 도와주는가"**다.

- A의 용신 오행을 B의 사주가 강화해주면 → A에게 좋은 인연
- B의 용신 오행을 A의 사주가 강화해주면 → B에게 좋은 인연
- 상호 용신을 강화해주면 → 최상의 궁합
- 용신을 기신으로 만들면 → 관계는 있지만 소모적

### 4단계: 지지 합충형파해 분석

두 사람의 **지지(地支) 8자 전체**를 놓고 관계를 본다.

**합(合)**: 육합, 삼합, 방합 → 결속, 인연의 끌림  
**충(沖)**: 자오충, 묘유충 등 → 충돌, 자극, 변화  
**형(刑)**: 인사신, 축술미 등 → 갈등, 상처  
**파(破)**: → 관계 균열  
**해(害)**: → 방해, 억제  

> 충이 무조건 나쁜 건 아니다. 양쪽 모두의 용신이 충으로 발동되면 오히려 활성화될 수 있다.

합충 분석 시 **중요도 순서**: 일지(배우자궁) > 월지 > 년지·시지

### 5단계: 대운 흐름 고려 (옵션)

현재 대운이 알려진 경우 추가 분석한다.

- 현재 각자의 대운이 관계에 미치는 영향
- 인연이 강화되는 시기 vs. 갈등이 커지는 시기
- 장기적 흐름에서 관계의 지속 가능성

## 출력 구조

인원 수에 따라 출력 구조를 아래와 같이 선택한다.

---

### [2인] 표준 궁합 출력

```
## [A] × [B] 궁합 분석

### 1. 두 사람의 사주 개요
- A: 일간 오행, 신강/신약 추정, 용신 추정
- B: 일간 오행, 신강/신약 추정, 용신 추정

### 2. 일간 오행 관계
- 생극 관계 + 의미

### 3. 용신 교환 분석
- A 입장에서 본 B / B 입장에서 본 A

### 4. 지지 합충형파해
- 주요 관계 열거 + 실질적 영향 (일지 우선)

### 5. 종합 해석
- 관계의 본질적 성격 / 강점·약점 / 적합한 관계 유형

### 6. 실용 조언
- 주의할 점 / 시너지 방법 / 갈등 패턴과 대처
```

---

### [3~4인] 다인 궁합 출력

```
## [그룹명 또는 인원 나열] 다인 궁합 분석

### 1. 개인별 사주 개요
- 각자의 일간, 오행 특성, 추정 용신을 표 형태로 정리

### 2. 쌍별(Pair) 궁합 요약
- A×B, A×C, B×C ... 모든 조합을 간략하게 정리
- 각 쌍: 일간 관계 + 핵심 합충 + 한줄 평가

### 3. 그룹 전체 역학 분석
- 그룹의 오행 분포 (어떤 오행이 강하고 약한가)
- 누가 그룹의 중심 역할을 하는가 (용신 제공자)
- 잠재적 갈등 구도 (충·형이 집중되는 관계)
- 그룹의 강점과 취약점

### 4. 관계 지도 (텍스트)
- 인물 간 관계를 텍스트로 표현
  예) A → B (생, 긍정) / B ↔ C (충, 긴장) / A — C (중립)

### 5. 종합 조언
- 그룹으로서 시너지를 내는 조건
- 갈등을 최소화하기 위한 역할 배분
- 특정 인물 간 주의 관계 명시
```

---

### [5인 이상] 대규모 그룹 출력

쌍별 분석은 **핵심 관계(갈등·시너지 TOP 3)**만 다루고,
나머지는 그룹 역학·오행 분포·역할 구도 중심으로 압축 출력한다.

---

## 해석 원칙

1. **사주첩경 우선**: 이석영의 이론을 1차 기준으로 적용. 현대 명리 보완 시 `(현대 명리)` 명시
2. **격국에서 출발**: 개인 분석은 월지 격국을 먼저 확인한 후 용신을 잡는다
3. **단정을 피한다**: "반드시 맞다/안 맞다" 대신 "～하는 경향이 있다", "～할 가능성이 높다"
4. **합충은 용신 맥락에서**: 합이 무조건 좋고 충이 무조건 나쁜 게 아니다. 생성되는 오행이 용신/기신인지 확인
5. **일주 중심**: 사주첩경 원칙대로 일지(배우자궁)를 궁합의 핵심으로 본다
6. **사람을 평가하지 않는다**: 사주 구조를 분석하는 것이지 인격을 평가하는 게 아님
7. **현실적 관점 유지**: 사주는 경향성이지 운명이 아님을 분석 말미에 명시
