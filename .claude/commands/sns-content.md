# SNS 콘텐츠 생성기

제3의시간 프로젝트의 SNS 콘텐츠를 생성합니다.

## 플랫폼

### 인스타그램 (주 3회)
- **화요일: 릴스** — 15~30초 분량 스크립트 + 캡션 + 해시태그
- **목요일: 캐러셀** — 슬라이드 5~7장 텍스트 + 캡션 + 해시태그 (장수/구성 변경 가능)
- **일요일: 릴스** — 15~30초 분량 스크립트 + 캡션 + 해시태그

### 스레드 (Threads)
- 인스타와 같은 주제의 텍스트 버전
- 500자 이내, 대화체

## 콘텐츠 소스

1. `content/blog/` 폴더의 블로그 글 (복길의 서고)
2. 사주/오행/육효점 관련 명리학 지식
3. 시즌 키워드 (월별 운세, 절기 등)

## 톤앤매너

- 브랜드: **제3의시간**
- 캐릭터: **복길** (마법사/도인, 친근하지만 신비로운 말투)
- 타겟: 20~30대 사주에 관심 있는 사람
- 금지: 길흉 단정, 특정 학자명, "운명은 정해져 있다" 류 표현
- 권장: 비유, 자기이해 관점, "알면 대비할 수 있다" 프레이밍
- CTA: 자연스럽게 서비스 유도 (ttt.betterdan.net)

## 해시태그 풀

```
기본: #제3의시간 #사주 #사주풀이 #오행 #명리학 #운세
확장: #사주팔자 #격국 #용신 #대운 #세운 #십성 #육효점
감성: #오늘의운세 #자기이해 #타고난기질 #시간의마법사 #복길
타겟: #MBTI보다사주 #사주입문 #사주공부 #사주스타그램
```

## 사용법

```
/sns-content 이번 주 콘텐츠 (4월 넷째 주)
/sns-content 릴스: 甲木 일간 성격
/sns-content 캐러셀: 오행이란?, 7장, 구성: 표지/질문/비유/상생/상극/정리/CTA
/sns-content 스레드: 대운이 바뀔 때
```

캐러셀은 호출 시 장수와 구성을 자유롭게 지정 가능합니다.
지정하지 않으면 기본 구성(표지/핵심/비유/심화/정리/CTA 6장)을 사용합니다.
