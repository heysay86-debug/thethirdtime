## 충 (沖) — 충돌·자극·변화
| 충 |
|----|
| 子午충 |
| 丑未충 |
| 寅申충 |
| 卯酉충 |
| 辰戌충 |
| 巳亥충 |

**궁합에서의 충 해석:**
- 일지 충(배우자궁): 가장 강한 충격. 갈등·이별 가능성.
- 단, 충이 용신 발동으로 작용하면 자극이 성장이 될 수 있음.
- 나이 차가 6년(같은 충 관계)이면 일지 충 가능성 높음.

---

## 형 (刑) — 갈등·상처·고집

### 삼형 (三刑)
- 寅巳申 (무은지형 — 은혜를 모르는 형)
- 丑戌未 (지세지형 — 권세를 믿는 형)

### 자형 (自刑) — 스스로를 해함
- 辰辰, 午午, 酉酉, 亥亥

### 무례지형
- 子卯형

**궁합에서의 형 해석:**
- 서로 자극하고 상처를 주는 관계
- 특히 일지나 월지에서 형이 겹칠 때 주의
