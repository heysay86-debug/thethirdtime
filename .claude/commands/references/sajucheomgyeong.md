# 이석영 사주첩경 — 핵심 이론 참조

> 이 스킬의 이론적 기반. 분석 시 이 원칙들을 우선 적용하고,
> 현대적 보완이 필요한 부분은 별도 표시한다.

---

## 1. 격국론 (格局論) — 사주첩경의 핵심

사주첩경은 **격국**을 사주 해석의 출발점으로 삼는다.
격국은 월지(月支)를 중심으로 결정되며, 사주의 전체 방향성을 규정한다.

### 내격 (內格) — 8격
| 격 | 조건 | 특성 |
|----|------|------|
| 정관격 (正官格) | 월지에 정관이 투출 | 명예, 질서, 사회적 규범 |
| 편관격 (偏官格) | 월지에 편관이 투출 | 추진력, 개혁, 승부욕 |
| 정인격 (正印格) | 월지에 정인이 투출 | 학문, 명예, 보수적 안정 |
| 편인격 (偏印格) | 월지에 편인이 투출 | 독창성, 예술성, 비정통 |
| 식신격 (食神格) | 월지에 식신이 투출 | 창의, 여유, 생산성 |
| 상관격 (傷官格) | 월지에 상관이 투출 | 재능, 반골기질, 예술 |
| 정재격 (正財格) | 월지에 정재가 투출 | 현실감각, 성실, 축재 |
| 편재격 (偏財格) | 월지에 편재가 투출 | 사업수완, 활동성, 인복 |

### 외격 (外格) — 특수격
- **종격(從格)**: 일간이 극히 약해 오행의 흐름에 따르는 사주
  - 종아격, 종재격, 종관격, 종살격, 종강격
- **화격(化格)**: 천간합이 이루어져 새로운 오행으로 변화
- **전왕격(專旺格)**: 한 오행이 극단적으로 강해 독주하는 사주

> **궁합 적용**: 두 사람의 격국이 서로 보완적인지 확인한다.
> 예) 상관격 × 정관격 → 긴장·갈등 구조 (상관이 정관을 극)
> 예) 식신격 × 편재격 → 시너지 (식신생재)

## 2. 용신론 (用神論) — 사주첩경 5가지 용신

사주첩경은 용신을 단일 방법으로 결정하지 않고, **5가지 방법론**을 순서대로 적용한다.

### ① 억부용신 (抑扶用神) — 가장 기본
- 신강 → 일간을 억제하는 오행 (관성·재성·식상)
- 신약 → 일간을 도와주는 오행 (인성·비겁)

### ② 조후용신 (調候用神) — 계절 균형
- 여름(巳午未월) 사주 → 壬癸水 (냉각)
- 겨울(亥子丑월) 사주 → 丙丁火 (온기)
- **사주첩경은 조후를 억부보다 우선시하는 경우가 많다**

### ③ 통관용신 (通關用神)
- 두 오행이 대립(상극)할 때 중간을 연결하는 오행
- 예) 木과 土의 극 → 火가 통관 (목생화, 화생토)

### ④ 병약용신 (病藥用神)
- 사주의 병(과도하거나 해로운 오행)을 제거하는 오행
- 사주첩경에서 특히 강조: "사주에 병이 있으면 약을 찾으라"

### ⑤ 전왕용신 (專旺用神)
- 종격·전왕격에서는 억부 반대로 적용 — 강한 기운을 따름

| 십신 | 일간 기준 | 사주첩경적 해석 |
|------|---------|--------------|
| 비견(比肩) | 같은 오행·양 | 경쟁자, 동료, 독립심 |
| 겁재(劫財) | 같은 오행·음 | 탈재, 경쟁, 반발 |
| 식신(食神) | 일간이 생, 같은 음양 | 재능 발현, 여유, 수명성 |
| 상관(傷官) | 일간이 생, 다른 음양 | 창의성, 반골, 구설 |
| 정재(正財) | 일간이 극, 다른 음양 | 안정적 재물, 배우자(남) |
| 편재(偏財) | 일간이 극, 같은 음양 | 유동 재물, 사업, 부친 |
| 정관(正官) | 일간을 극, 다른 음양 | 명예, 규범, 배우자(여) |
| 편관(偏官) | 일간을 극, 같은 음양 | 권력, 도전, 스트레스 |
| 정인(正印) | 일간을 생, 다른 음양 | 학문, 모친, 안정 |
| 편인(偏印) | 일간을 생, 같은 음양 | 독창성, 고독, 편법 |

## 4. 궁합에서 사주첩경이 강조하는 원칙

### 원칙 1: 일주(日柱) 중심
> "궁합은 일주로 본다" — 사주첩경

일지(배우자궁)의 합충이 궁합의 핵심. 년주·월주는 참고 수준.

### 원칙 2: 용신 제공 여부
상대방이 나의 용신 오행을 가지고 있으면 인연이 있다고 본다.
단순 오행 상생이 아니라 **나의 필요(용신)**를 채워주는지가 기준.

### 원칙 3: 합이 많다고 좋은 게 아니다
합이 기신 오행을 생성하면 오히려 흉. 합의 화(化) 오행이 용신인지 기신인지 반드시 확인.

### 원칙 4: 충은 동(動)이다
충이 있다고 나쁜 게 아니라 "움직임·변화"를 의미한다.
일지 충 = 배우자궁의 충격. 갈등일 수도 있고 서로 자극하는 활성화일 수도 있다.

### 원칙 5: 형(刑)은 세심히 본다
사주첩경은 형을 합·충보다 미묘하게 다룬다.
형은 외적 충돌보다 **내면의 갈등, 상처, 억압**으로 나타나는 경우가 많다.

## 5. 현대적 보완 사항 (사주첩경 이후)

사주첩경은 전통 고전 이론에 충실하지만, 현대 명리 연구에서 보완된 부분들이 있다.
분석 시 아래 사항은 현대적 시각을 함께 제시한다.

| 주제 | 사주첩경 원칙 | 현대적 보완 |
|------|------------|-----------|
| 여성 사주 | 관성 = 남편, 재성 = 시댁 (전통적) | 현대는 여성 독립·커리어 맥락 추가 적용 |
| 편관(七殺) | 흉신으로 보는 경향 | 현대는 제살(制殺)되면 추진력·리더십으로 재해석 |
| 상관견관 | 매우 흉 (관재·이혼) | 현대는 맥락에 따라 직업 전환·창업 에너지로 봄 |
| 종격 판단 | 엄격한 기준 | 현대는 "가종격(假從格)" 개념 추가 — 완전하지 않은 종격 |
| 대운 해석 | 천간·지지 각각 5년 | 현대는 전체 10년을 통합해서 봄 |

> **표시 원칙**: 현대적 보완을 쓸 때는 괄호로 `(현대 명리)` 라고 명시하여
> 사주첩경 원칙과 구분한다.
