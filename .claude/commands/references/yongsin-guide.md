# 용신 추론 가이드라인

> 용신은 사주 전체를 봐야 정확하게 잡힌다. 이 파일은 궁합 분석 시 빠르게 추론하기 위한 가이드다.
> "추정 용신"으로 표현하고, 단정 짓지 않는다.

## 1단계: 일간의 강약 판단 (신강/신약)

### 득령(得令) 여부 — 월지와의 관계
| 일간 오행 | 왕지(月支) |
|----------|-----------|
| 木 왕 | 寅卯(봄), 亥(수생목) |
| 火 왕 | 巳午(여름), 寅卯(목생화) |
| 土 왕 | 辰戌丑未(환절기) |
| 金 왕 | 申酉(가을), 辰戌丑未(토생금) |
| 水 왕 | 亥子(겨울), 申酉(금생수) |

### 득지(得地) 여부 — 일지와의 관계
- 일지가 일간과 같은 오행이거나 생해주면 득지

### 득세(得勢) 여부 — 천간·지지 전체
- 나와 같은 오행(비겁), 나를 생하는 오행(인성)이 많으면 신강

### 신강/신약 판단 요약
- **신강**: 월지+기타에서 일간 오행이 강함 → 설기(食傷), 재성, 관성이 용신 후보
- **신약**: 일간이 약함 → 비겁, 인성이 용신 후보

---

## 2단계: 계절(월지)별 용신 경향

| 월지 | 계절 | 일반 용신 경향 |
|------|------|--------------|
| 寅卯辰 | 봄 | 목 왕성 → 화(설기) 또는 금(극목)이 조후/용신 |
| 巳午未 | 여름 | 화 왕성·건조 → 수(조후) 중요 |
| 申酉戌 | 가을 | 금 왕성 → 화(극금) 또는 목 |
| 亥子丑 | 겨울 | 수 왕성·한랭 → 화(조후) 중요 |

**조후(調候) 우선**: 여름 사주는 壬癸水, 겨울 사주는 丙丁火가 조후용신으로 우선시됨.

---

## 3단계: 궁합에서 용신 교환 해석

```
A의 추정 용신: [오행]
→ B의 사주에서 해당 오행이 강한가? (천간에 투출 or 지지에 많음)
  YES → B는 A에게 좋은 영향 (길한 인연)
  NO, 오히려 기신 오행이 강함 → 소모적 관계

B의 추정 용신: [오행]
→ A의 사주에서 해당 오행이 강한가?
  YES → A는 B에게 좋은 영향
  NO → A가 B에게 부담
```

---

## 주의사항

- 이 가이드는 빠른 추론용. 실제 용신은 격국(格局), 조후, 병약론 등 복합 이론으로 결정됨.
- "추정 용신"이라고 명시하고 사용자에게 오차 가능성을 알린다.
- 용신을 알 수 없는 경우, 오행 생극 관계와 합충만으로도 궁합 분석은 가능하다.
