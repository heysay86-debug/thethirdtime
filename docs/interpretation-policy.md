# 해석 정책 — 이석영 기준 + 5유파 보강

최종 업데이트: 2026-04-15

---

## 기본 원칙

sajuweb의 해석 출력은 **2계층 구조**를 따른다.

1. **1차 판단 (계산·판정 레이어)**: 이석영 『사주첩경』 기준
   - 격국 판단, 용신 도출, 신강/신약 판정은 반드시 이 기준으로 먼저 확정한다.
   - 이 결과가 사주 해석의 뼈대이며, 타 유파와 충돌 시 이 기준이 우선이다.

2. **2차 보강 (해석·서술 레이어)**: 5개 유파 현대적 해석 병기
   - 1차 결과를 바탕으로, 각 유파의 관점을 추가 서술하여 해석의 깊이를 더한다.
   - 유파 간 견해 차이가 있을 경우 차이점을 명시하고 독자가 판단하도록 제시한다.
   - 1차 결론을 뒤집거나 모순을 일으키는 방식으로 사용하지 않는다.

## 출력 구조 (Claude 프롬프트 설계 기준)

### 개인 사주 분석 출력 구조

```
[1] 사주팔자 제시
    연·월·일·시주 간지 (한자 + 오행)

[2] 핵심 판단 (이석영 기준)
    - 일간 강약 (신강/신약)
    - 격국
    - 용신 / 기신

[3] 주별 심층 분석
    - 연주, 월주, 일주, 시주 각각의 의미
    - 지장간, 십이운성

[4] 오행 분석
    - 오행 분포 및 과부족
    - 조후 고려 (궁통보감 관점 병기)

[5] 십성 분석
    - 십성별 배치 해석
    - 육친 관계 (명리정종 관점 병기)

[6] 형충파해합·신살

[7] 대운 흐름

[8] 종합 해석
    - 이석영 기준 종합
    - 유파별 보강 관점 (필요 시):
      "적천수 관점에서는 ~", "궁통보감 기준 조후로 보면 ~" 형식으로 병기
    - 현대적 적용 (직업·성향·관계 등)
```

## 유파 보강 사용 원칙

1. **병기 표시 명확화**: 유파 보강 내용은 반드시 출처를 밝힌다.
   예: "적천수 관점에서는 일간 기세를 우선하여 ~로 볼 수 있다."

2. **1차 결론 보호**: 2차 보강이 1차 결론(格局·用神)을 부정하는 방식으로 사용하지 않는다.
   예) "이석영 기준 관격이지만 적천수 기준으로는 인수격일 수 있다" 같은 혼란 유발 서술 금지.

3. **해석 단계에서만 활용**: 계산 로직(4기둥, 절입 기준, 지장간 비율 등)에는 이석영 기준만 적용.

4. **이론 근거 없는 즉흥 해석 금지**: "~할 수도 있다", "~인 경향이 있다" 식의 근거 없는 서술 금지.
   모든 해석에 어느 이론의 어느 원칙에 근거하는지가 명확해야 한다.

## Claude API 연동 방식

### 시스템 프롬프트 (역할·원칙·서술 지침만)

시스템 프롬프트에는 해석 역할, 이석영 1차 기준, 5유파 보강 원칙, 서술 지침만 포함한다.
**출력 JSON 스키마는 시스템 프롬프트에 기술하지 않는다.**

상세 프롬프트: `src/gateway/prompts/system.ts`

### 출력 구조 강제: Anthropic Tool Use

출력 JSON 스키마는 Anthropic tool use(function calling) 정의로 분리하여
LLM이 반드시 지정된 구조로만 응답하도록 강제한다.

- Tool 정의: `src/gateway/tools/saju_interpretation.ts`
- Zod 스키마: `src/gateway/prompts/schema.ts` → `InterpretationResultSchema`
- 참고: https://docs.anthropic.com/en/docs/agents-and-tools/tool-use/overview

### Prompt Caching

시스템 프롬프트와 tool 정의는 매 요청마다 동일하므로,
`cache_control: {"type": "ephemeral"}`을 지정하여 반복 호출 시 토큰 비용을 절감한다.

- 캐시 TTL: 5분 (Anthropic 기본값)
- 캐시 히트율 측정: `usage.cache_creation_input_tokens`, `usage.cache_read_input_tokens` 로깅
- 참고: https://docs.anthropic.com/en/docs/build-with-claude/prompt-caching

### 프롬프트 핵심 지침 (요약)

```
당신은 이석영 『사주첩경』을 1차 기준으로 삼는 사주 분석 전문가입니다.
격국·용신·신강신약 판단은 엔진이 계산한 값을 존중하십시오.
해석 서술 단계에서는 자평진전·적천수·궁통보감·명리정종 관점을 보조로 활용하되,
출처를 명시하고 1차 결론과 모순되지 않는 방식으로 병기하십시오.
이론 근거 없는 추측성 해석은 하지 마십시오.
```

## 유파별 주요 활용 포인트

각 유파를 어느 분석 항목에 주로 활용할지 가이드.

- **자평진전**: 격국의 성패(成格·破格) 판단 보강. "이 격국이 성격인지 파격인지 자평진전 기준으로도 ~"
- **적천수**: 일간 기세와 용신의 현대적 재해석. 특히 종격(從格) 판단 시 참조.
- **궁통보감**: 조후 분석. 계절·월령에 따른 일간 보완 필요 여부. "동절 수일간으로 조후상 화가 필요하며 ~"
- **명리정종**: 십성 위주 육친 분석. 부모·배우자·자녀 관계 서술 시 참조.
