# 05. 주별 심층 분석 보강 가이드라인
**연주 · 월주 · 일주 · 시주 각각의 의미**

---

## 섹션 목표

현재 상태: 조견표에 간지 표시 + 1-2문장의 기본 설명

개선안: 각 주가 **인생의 어느 시기를 주도하며, 어떤 역할을 하는지** 구체적으로 설명

---

## 전체 구조 (4개 주 × 300~350글자 = 약 1,200~1,400글자)

```
각 주별로:

[기본 정보]
├ 간지 (한자 + 읽기)
├ 오행 분류
├ 십성 분류
└ 인생 주기상 의미

[지장간 설명]
├ 지장간이 무엇인지
├ 현재 지장간 구성
└ 투출 여부 확인

[현재 사주와의 역할]
├ 전체 사주에서의 위치
├ 다른 주와의 관계 (형충파해)
└ 대운 변화에 따른 영향

[에너지 특성]
├ 긍정적 측면
├ 주의할 점
└ 현대 적용
```

## 각 주별 상세 구성

### 패턴 (각 주 300~350글자)

#### 구성 1: 기본 정보 (50~70글자)
```
연주 丙寅(병인)은 당신의 인생에서 [어린 시절]을 상징합니다.
병화는 태양의 밝음과 따뜻함을, 인목은 새로운 생명과 성장을 나타냅니다.
```

#### 구성 2: 오행 구조 + 지장간 (80~120글자)
```
지장간으로 목·화·토를 품고 있으며, 목(정기)과 화(차기)가 투출되어 있습니다.
이는 이 시기의 에너지가 복합적이면서도 명확하게 표현된다는 의미입니다.
목은 성장의 욕구를, 화는 그 욕구를 적극적으로 드러내는 행동으로 변환합니다.
```

#### 구성 3: 십성 분석 (60~100글자)
```
연주의 십성을 보면 [편재], [식신] 등이 배치되어 있습니다.
이는 당신의 어린 시절이 활동적이었으며, 많은 관심과 호기심이 있었음을 암시합니다.
또한 외부 세계와의 상호작용이 많았을 가능성이 높습니다.
```

#### 구성 4: 인생주기와의 연결 (60~100글자)
```
연주는 보통 0~15세를 관장하며, 이 시기 당신은 호기심 많은 아이였을 것입니다.
병인의 밝음과 활동성 때문에, 정적인 환경보다는 움직이고 배우는 환경에서 
성장했을 가능성이 높습니다.
```

#### 구성 5: 현재 사주와의 관계 (60~100글자)
```
연주의 병화와 일주의 임수는 [수극화] 관계이므로, 표면적으로 마찰이 있어 보일 수 있습니다.
하지만 이 마찰이 당신에게 균형을 주는 것입니다. 
어린 시절의 활발함이 중년의 신중함으로 보완되는 구조입니다.
```

## 구체 예시: 연주 (丙寅) 완성본

```
연주 丙寅(병인)은 당신의 인생에서 [어린 시절과 청소년기]를 상징합니다.
병화는 태양의 밝음을 의미하고, 인목은 봄의 새로운 생명을 나타냅니다.

지장간으로 목·화·토를 품고 있으며, 이 중 목과 화가 명확하게 투출되어 있습니다.
이는 이 시기 당신의 에너지가 외부로 뚜렷하게 드러났음을 의미합니다.
목은 성장하려는 욕구, 화는 그 욕구를 행동으로 표현하는 능력으로 작용합니다.

연주의 십성을 살피면 [편재], [식신] 등이 있어, 당신의 어린 시절은 
외부 활동이 많고, 관심사가 다양했을 것으로 추측됩니다.
규칙적인 학습보다는 체험을 통한 학습이 더 효과적이었을 가능성이 높습니다.

일주의 임수와 연주의 병화는 [수극화] 관계입니다.
표면적으로는 마찰처럼 보이지만, 이 마찰이 당신에게 균형을 주는 것입니다.
어린 시절의 적극적 행동성이 성인이 되면서 사려 깊은 신중함으로 발전하게 되는 
과정이 이 구조에서 나타나는 것입니다.
```

## 4개 주의 인생주기 가이드

| 주 | 인생주기 | 의미 |
|---|---|---|
| **연주** (Year Pillar) | 0~15세 (어린시절·청소년) | 타고난 기질, 가정환경의 영향 |
| **월주** (Month Pillar) | 16~30세 (청년·초반 중년) | 사회 진출, 기본 성격 형성 |
| **일주** (Day Pillar) | 31~55세 (중년) | 결혼, 가족, 자아 확립 |
| **시주** (Hour Pillar) | 56~80세+ (노년) | 자식, 유산, 인생 마무리 |

> 주: 개인차가 크므로 참고만 하고, 대운 변화와 함께 해석할 것

## 형충파해합과의 연결 표현

각 주 설명에서 다른 주와의 관계를 명시:

```
[안 좋은 예]
연주와 월주가 형을 이룹니다.

[좋은 예]
연주의 병인과 월주의 정유는 [형(刑)] 관계로, 어린 시절의 활발함과 
청소년기 독립심이 때로 충돌하는 양상을 보입니다.
하지만 이 충돌이 성장의 기회가 되어, 당신을 더욱 단단하게 만들었을 것입니다.
```

## 체크리스트 (각 주별)

- [ ] 간지명과 의미(한자)를 함께 설명했는가?
- [ ] 지장간 구성과 투출 여부가 명확한가?
- [ ] 십성까지 언급하여 복합적 이미지가 그려지는가?
- [ ] 인생주기(연령대)와 사주의 에너지를 연결했는가?
- [ ] 다른 주와의 형충파해 관계를 긍정적으로 해석했는가?
- [ ] 조사가 자연스럽게 이어지는가?
- [ ] 한 주당 300~350글자 범위 내인가?

## 작성 팁

1. **인생주기를 먼저 명시**: "연주는 어린 시절을 의미합니다"로 시작하면 일반인도 쉽게 이해
2. **지장간을 "계절"과 연결**: "봄은 만물이 움트는 시기이므로..." 방식으로 설명
3. **십성을 생활 수준으로**: "편재는 불규칙한 수입을 의미합니다" → "외부 활동이 많다"
4. **형충파해를 성장담으로**: "마찰=나쁨" 대신 "마찰=도전=성장" 프레임
5. **현재 대운과의 비교**: "현재 당신은 00대운 중이므로..." 방식으로 즉시성 제공
