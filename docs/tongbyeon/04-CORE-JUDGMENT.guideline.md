# 04. 핵심 판단 보강 가이드라인
**신강/신약 · 격국 · 용신 설명**

---

## 섹션 목표

현재 상태: 신강도 수치 + 격국명 + 용신 오행 (기본 정보만)

개선안: 일반인이 이해 가능한 수준으로 **자신의 기본 성격 틀**을 설명

---

## 전체 구조 (약 1,200~1,500글자)

```
[1] 신강/신약 설명 (300~350글자)
    ├ 개념 정의 (한자 + 뜻)
    ├ 오행 상관관계 (예: 토일간 + 월령)
    ├ 현재 수치 의미
    └ 성격 특성 대비

[2] 격국 설명 (300~350글자)
    ├ 격국명 + 의미
    ├ 해당 격국이 나타나는 구조
    ├ 성공 조건 vs 실패 조건 (성격 vs 파격)
    └ 현재 사주에서의 위치

[3] 용신 설명 (300~350글자)
    ├ 용신이 무엇인지 (일간을 돕는 오행)
    ├ 왜 필요한지 (신강약 보정)
    ├ 용신의 역할 (인생의 조력자)
    └ 이 사주에서 용신의 표현 방식
```

---

## 섹션 1: 신강/신약 (약 300~350글자)

### 구성

#### 1-1. 개념 정의 (50~70글자)
```
신강(身强)과 신약(身弱)이란 용어는 일간의 기운이 월령을 중심으로 
태어난 계절에 얼마나 강한지, 약한지를 판단하는 개념입니다.
```

#### 1-2. 오행 상관관계 설명 (80~120글자)
- 어떤 오행이 강하게 작용하고
- 월령(계절)이 어떤 역할을 하며
- 천간·지지의 지장간이 어떻게 배합되는지

```
예) 토일간이 하절월(오월)에 태어났다면, 태양의 열기(화)가 토를 건조시키므로 
토가 강해집니다. 다만 토의 습성(습토)에 따라 달라지므로...
```

#### 1-3. 현재 수치 의미 (50~80글자)
```
당신의 신강도는 70점입니다. 이는 기준점 50점을 훨씬 웃도는 수준으로,
일간이 충분히 견고한 기초를 가지고 있다는 의미합니다.
```

#### 1-4. 성격 특성 대비 (100~150글자)
```
신강의 경우 자기주장이 명확하며, 독립심이 강합니다.
반면 타인의 의견을 수용하기보다는 자신의 판단을 우선하는 경향이 있습니다.
다만 신강이 극단으로 가면 고집이 되고, 적절히 조절되면 리더십으로 발현됩니다.
```

## 섹션 2: 격국 (약 300~350글자)

### 구성

#### 2-1. 격국명 + 기본 의미 (60~80글자)
```
당신의 사주는 [편인격(偏印格)]입니다. 편인격이란 월주의 지지에 있는 
편인이 일간을 생해주고, 이것이 격을 이루는 구조를 말합니다.
```

#### 2-2. 격국이 나타나는 구조 설명 (80~120글자)
```
지지 유금에는 신금·경금이 지장간으로 있으며, 이 중 경금이 당신의 시주 천간에 
투출되어 있습니다. 이렇게 지지의 기운이 천간으로 드러나면서 격을 형성하게 됩니다.
```

#### 2-3. 격국의 성격 특성 (100~150글자)
```
편인격은 정통 경로보다 파격적 접근을 선호합니다. 학문이나 기술도 
정석보다는 자신만의 방식을 개발하는 데 능합니다.
하지만 이는 변덕으로 보일 수 있으며, 일관성을 잃기 쉽습니다.
현대에는 이 특성이 창의성·다재다능함으로 평가받습니다.
```

#### 2-4. 성공 조건 (성격 vs 파격) (80~120글자)
```
편인격이 성공하려면 자신의 파격적 사고를 현실에 구현할 수 있는 
실행력이 필요합니다. 또한 한 우물을 파되, 그 안에서 혁신을 이루는 
전략이 도움이 됩니다.
```

## 섹션 3: 용신 (약 300~350글자)

### 구성

#### 3-1. 용신 개념 (60~80글자)
```
용신(用神)이란 일간의 기운을 돕고 조화시키는 오행을 말합니다.
당신의 경우 [목(木)]이 용신으로, 임수 일간을 자극하고 
통장(通暢)하게 만드는 역할을 합니다.
```

#### 3-2. 왜 필요한지 (오행 상관관계) (80~120글자)
```
임수는 흐르는 물의 기운입니다. 물이 적절히 흐르려면 나무라는 통로가 필요합니다.
(수생목의 역 관계에서) 목이 물의 흐름을 조절하고 방향을 잡아주는 것과 같습니다.
따라서 목의 기운이 충분해야 당신이 자신의 기운을 온전히 발휘할 수 있습니다.
```

#### 3-3. 용신의 사주 내 표현 (80~120글자)
```
당신의 사주에서 목은 연주 인목과 지지에 잠긴 여러 목 기운으로 나타납니다.
다만 전체 오행분포에서 목의 비중이 상대적으로 적은 편이므로,
대운이나 년운에서 목이 충분하게 들어올 때 당신의 잠재력이 가장 잘 발현됩니다.
```

#### 3-4. 용신 부족 시 보조 (기신, 희신) (60~100글자)
```
용신이 약할 때는 기신(기가 되는 신)이 그 역할을 합니다.
당신의 경우 [화(火)]가 기신 역할을 하여, 목을 북돋아주는 역할을 합니다.
```

---

## 예시: 편인격 + 신강 + 목 용신 통합 설명

```
당신의 사주는 신강도 70점으로 자기주장이 명확하고 독립심 강한 기초를 가졌습니다.
이 강한 기운이 격국으로는 [편인격]을 이루는데, 편인격은 정통보다 파격적 
경로를 선호하는 특징이 있습니다. 따라서 당신은 자신의 방식을 개발하는 데 능하지만,
그 과정에서 일관성을 잃거나 변덕스러워 보일 수 있습니다.

이 강한 의지와 파격성을 조화시키는 것이 바로 [목 용신]입니다.
목은 물을 흐르게 하는 통로와 같아서, 당신의 기운을 올바른 방향으로 
흐르도록 안내합니다. 현재 당신의 사주에서 목이 충분하지 않은 편이므로,
목 기운이 많은 시기(대운/년운)에 당신의 창의성과 혁신성이 가장 빛날 것입니다.
```
