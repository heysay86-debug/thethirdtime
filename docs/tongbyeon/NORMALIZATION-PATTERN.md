# 사주 해석 텍스트 정규화 패턴
**sajuweb 일반인 친화형 보강 스크립트 표준**

최종 업데이트: 2026-04-17

---

## 개요

사용자가 작성한 "십성 분석 보강 스크립트"를 분석하여 추출한 **메타 패턴**.
모든 목차(섹션)의 LLM 출력에 동일하게 적용 가능한 구조.

---

## 핵심 원칙 (3가지)

### 1. 조사 자연스러움 (문법적 연속성)
**목표**: "-이/-가" 반복을 제거하고, 문장 흐름을 자연스럽게

**방법**:
- 주어를 명확히 → "이를", "이는", "이에 속합니다" 활용
- 관형사형 종결 → "~은 의미입니다" (X) → "~를 의미합니다" (O)
- 나열 시 "이 중", "다른 한편", "반면" 등으로 자연스럽게 이어짐

**예시**:
```
[X] 십성은 관계성이다. 십성은 대비된다. 십성은 의미를 갖는다.
[O] 십성은 일간의 기운을 기준으로 자신을 생하고 극하는 관계성이다. 
    이 관계성을 음양의 일치성으로 분화하면 다섯 가지 유형을 10개로 표현할 수 있다.
```

---

### 2. 전문 용어 3단계 풀이
**목표**: 일반인이 이해 불가능한 용어를 체계적으로 설명

**3단계 구조**:

| 단계 | 내용 | 예시 |
|---|---|---|
| **1단계** | 한자 읽기 + 기본 정의 | "투출(投出)이란 지지 안의 오행이 천간으로 표면에 드러나는 것" |
| **2단계** | 개념의 근원 또는 비유 | "지지는 계절의 순환을 의미하며, 계절은 서서히 변한다..." |
| **3단계** | 구체 예시 (사주와 연관) | "일주 일간이 임수라면 천간에서 계수가 나타나면 겁재가 되고..." |

**응용 패턴**:
- **한자 의미 활용**: "재물을 빼앗는다" (겁재의 財字 이용)
- **일상 행동어**: "도장을 찍는 행위" (인성), "먹는 행위" (식신)
- **자연 현상**: "계절의 변화", "파도의 침식"
- **사회 역할**: "계약·문서", "평판·권한"

---

### 3. 균형잡힌 대비 서술
**목표**: 길흉론의 주관성 제거, 객관적 특성만 병기

**구조**:
```
[개념 설명]
장점 1, 장점 2를 의미합니다. 

[하지만/반면] 
단점 1, 단점 2의 가능성이 있습니다.

[컨텍스트]
따라서 현재 사주의 오행분포(또는 형충파해)에 따라 
이 에너지가 강해질 수도, 약해질 수도 있습니다.
```

**예시 (겁재)**:
```
겁재의 장점은 섬세함, 혁신적 사고입니다.
그러나 한자에서 유추할 수 있듯 재물을 빼앗는다는 의미이므로,
낭비벽·외부 에너지 발산이 문제될 수 있습니다.
```

## 섹션별 적용 패턴 (메타 구조)

### 패턴 A: 개념형 (십성, 오행, 신살)

**구성**:
1. **정의** (1-2문): 한자 + 기본 개념
2. **이론 근거** (2-3문): 오행 상관관계, 지장간, 음양 등
3. **메타포** (1-2문): 일상/자연 비유
4. **사주 내 위치** (1-2문): 해당 사주에서의 구체 위치
5. **특성 - 장점** (2-3문): 긍정 특성 + 예시
6. **특성 - 단점** (2-3문): 부정 특성 + 이유
7. **컨텍스트** (1-2문): 오행분포, 형충파해와의 상호작용

**길이**: 250~350글자 (십성 정도)

---

### 패턴 B: 분석형 (주별 분석, 격국)

**구성**:
1. **기본정보** (1-2문): 간지 + 오행 + 위치의미
2. **지장간 풀이** (2-3문): 지장간이 무엇이고 왜 중요한지
3. **현재 사주와의 역할** (2-3문): 이 주가 전체 사주에서 어떤 기능
4. **에너지 특성** (2-3문): 장단점 대비
5. **다른 주와의 상호작용** (1-2문): 형충파해, 생극 관계

**길이**: 300~400글자

---

### 패턴 C: 통합형 (대운, 종합해석)

**구성**:
1. **총괄 진단** (1-2문): 전체 이미지 한 문장
2. **주요 특징** (3-4문): 가장 중요한 2-3가지
3. **주기별 변화** (3-4문): 시간에 따른 변화 추세
4. **주의사항** (2-3문): 피해야 할 점, 활용 방법
5. **격려** (1-2문): 앞을 바라보는 톤

**길이**: 400~500글자

---

## 금지 사항 (체크리스트)

- [ ] "-이/-가" 반복 (문장 연속성 파괴)
- [ ] 전문 용어를 1단 정의만 제시하고 끝냄
- [ ] "할 수 있다", "경향이 있다" 식 근거 없는 추측
- [ ] 길흉론 (흉하다 O → 흉함을 의미한다 X)
- [ ] 1차 결론(격국·용신)을 뒤집는 2차 보강
- [ ] 이론적 근거 없이 특정 유파만 강조
- [ ] 오행분포·형충파해와 무관한 단편 해석

---

## 서술 스타일 가이드

### 톤
- **객관적**: "될 수 있습니다" 금지 → "의미합니다", "특징입니다"
- **따뜻함**: 경고조 금지 → "살피시면 좋겠습니다", "이해하시면..."
- **이석영 기준**: 1차 결론 명확 → "이석영 기준 ~입니다"

### 문장 길이
- 한 문장: 50~80글자 (일반인 가독성)
- 문단: 2-4문장 (호흡)
- 섹션: 250~500글자 (휴식 없이 읽기 어렵지 않은 수준)

### 표현 금기
| 금지 | 대체 |
|---|---|
| "~할 가능성이 높습니다" | "~의 특징을 보입니다" |
| "~이므로 주의하세요" | "~이므로 이를 이해하면 도움됩니다" |
| "불운할 시기" | "변화의 시기", "도전의 시기" |
| "성공/실패" | "성취/도전", "순탄/고난" |

## 예시 분석

### ✅ 좋은 예 (사용자 십성 스크립트 - 겁재)

```
지지 시주에 겁재가 존재합니다. 겁재는 일간과 같은 오행이지만 음양이 다른 글자입니다.
[→ 정의 명확, 조사 자연스러움]

일주 일간이 임수라면 천간에서 계수가 나타나면 겁재가 되고, 지지에서는 자수가 겁재가 됩니다.
[→ 구체 예시, 오행 상관관계 설명]

겁재의 장점은 섬세하다는 것, 그리고 파괴적이고 혁신적인 사고를 하는 힘입니다.
[→ 메타포 "파괴적·혁신적" = 기존을 깨는 힘]

그러나 한자에서 유추할 수 있듯, 이는 재물을 빼앗는다는 의미입니다.
[→ 한자 의미 활용]

낭비벽이 있을 수 있으며 외부인에게는 평판을 위해 애쓰지만 
반대급부로 에너지가 외부로 발산하기에 내부인에게는 소홀할 가능성이 높습니다.
[→ 장단점 대비, "하지만" 구조]
```

### ✅ 좋은 예 (주별 분석 - 보강안)

```
연주 丙寅(병인, 화목)은 이 사주의 에너지 시작점입니다.
병화는 태양으로, 따뜻함과 밝음을 의미하고, 인목은 새로운 생명을 상징합니다.
[→ 기본정보 + 메타포]

지장간으로 목·화·토를 품고 있으며, 이 중 목과 화가 투출되어 
겹겹이는 오행 강함을 드러냅니다.
[→ 지장간 설명 + 사주와의 연관]

따라서 당신의 어린 시절부터 중년까지는 이 밝음과 혁신성이 주도적 역할을 하였을 것입니다.
[→ 사주와의 직접 연관, 시간에 따른 변화 암시]
```

### ❌ 나쁜 예

```
연주는 성격이 밝습니다.
[→ 조사 반복, 근거 부족]

병화는 태양의 기운이고 인목은 나무인데, 이것은 불길한 특징을 의미합니다.
[→ 길흉론, "특징을 의미한다" 자체가 불명확]

지장간에 목화토가 있으므로 조심하세요.
[→ "조심하세요" 식 경고조, 근거 설명 부족]
```

## 적용 체크리스트

각 섹션 작성 후, 다음을 확인하세요:

- [ ] 전문 용어마다 한자 + 기본정의 + 구체예시 3단계?
- [ ] 문장이 "-이/-가"로 끝나지 않고 자연스럽게 이어짐?
- [ ] 장점과 단점이 "하지만", "반면" 등으로 균형있게 배치?
- [ ] 이 분석이 현재 사주의 오행분포·형충파해와 연관지어짐?
- [ ] 길흉론이 아닌 "특성 설명" 방식?
- [ ] 한 문단이 4문장 이상 6문장 이하?
- [ ] 전체 섹션이 250~500글자 범위 내?
