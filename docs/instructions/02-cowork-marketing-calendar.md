## 요청 작업

### 작업 1: 4주 런칭 캘린더 (.xlsx)

5/20 런칭 기준 4주(5/5~5/31) 캘린더를 만들어줘.

엑셀 구조:
- 시트 1 "캘린더": 날짜 | 요일 | 채널 | 콘텐츠 유형 | 제목/주제 | 카피 요약 | 상태(미작성/작성완료/게시완료) | 비고
- 시트 2 "채널별 요약": 채널별 총 게시물 수, 유형 분포
- 시트 3 "해시태그 세트": 콘텐츠 유형별 해시태그 조합 5세트

일정 구성 가이드:
- **1주차 (5/5~5/11)**: 프리런칭. 티저 콘텐츠, "곧 만나요" 분위기. 인스타 3회.
- **2주차 (5/12~5/18)**: 기대감 빌드업. 서비스 소개, 복길 캐릭터 소개. 인스타 3회 + 카카오 1회.
- **3주차 (5/19~5/25)**: **런칭 주간**. 5/20 런칭 당일 집중. 인스타 4회 + 카카오 2회 + 당근 1회 + 코인노래방 게시 시작.
- **4주차 (5/26~5/31)**: 후속. 후기/사례, 심화 콘텐츠. 인스타 3회 + 카카오 1회 + 당근 1회.

### 작업 2: 채널별 콘텐츠 문안 (.docx)

각 채널별 콘텐츠 문안을 작성해줘. 하나의 워드 파일에 채널별 섹션으로 구분.

#### 인스타그램 (12건)

캐러셀 4건:
1. 서비스 소개 ("RPG형 사주풀이가 뭔데?") — 8장
2. 사주 입문 교육 ("사주팔자, 4글자의 비밀") — 7장
3. 런칭 안내 ("5/20, 제3의시간이 열립니다") — 6장
4. 후기형 ("해봤는데... 소름이었다") — 7장

릴스 스크립트 8건:
1. 티저 ("운명을 읽는 마법사가 온다")
2. 복길 자기소개 ("내가 복길이일세")
3. "MBTI보다 깊은 자기이해" 비교
4. 사주 오행 간단 설명 (나는 무슨 오행?)
5. 런칭 카운트다운
6. 서비스 실제 화면 워크스루
7. 육효점 소개 ("점 한번 쳐볼까?")
8. "사주에서 가장 중요한 한 글자" (일간 설명)

#### 카카오 채널 (4건)

1. 프리런칭 안내 메시지 ("곧 만날 수 있을걸세")
2. 런칭 알림 ("드디어 열렸네, 자네의 시간이")
3. 첫 주 후속 ("벌써 OOO명이 다녀갔다네")
4. 콘텐츠 안내 ("이번 주 복길의 서고에 새 글이 올라왔네")

#### 당근마켓 (2건)

1. 로컬 홍보 ("OO동 근처에서 사주 보는 법")
2. 코인노래방 연계 ("노래방에서 사주를? 진짜입니다")

#### 코인노래방 (2건)

1. 벽보 카피 (A3 사이즈, QR코드 포함)
2. 대기화면 광고 카피 (16:9, 5초 내 훅)

## 톤앤매너 (반드시 준수)

- 복길 말투: 해라체+고풍 ("~일세", "~하겠네", "자네")
- 금지: 길흉 단정, 특정 학자명/서적명, "운명은 정해져 있다", "회원가입/로그인"
- 권장: 오행 비유, 자기이해 관점, RPG/게임 용어
