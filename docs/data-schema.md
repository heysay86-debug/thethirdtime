## iljin.json

### 구조
```json
{
  "YYYY-MM-DD": "한글(한자)",
  ...
}
```

### 샘플
```json
{
  "1900-01-01": "갑술(甲戌)",
  "1970-06-15": "병인(丙寅)",
  "2049-12-31": "경진(庚辰)"
}
```

### 포맷 주의
- 키: `"YYYY-MM-DD"` (모든 부분 zero-padded)
- 값: 한글 간지 + 괄호 안에 한자 간지
- 한자만 추출 시: `re.search(r'\((.+)\)', value).group(1)`
- 한글만 추출 시: `value.split('(')[0]`

### 커버리지
- 54,787개 날짜 (누락 0건 검증 완료)
- 2050년 이후 데이터 없음

## jeolip.json

### 구조
```json
{
  "YYYY": [
    {"name": "절기명", "month": int, "day": int, "time": "HH:MM"},
    ...
  ]
}
```

### 샘플 (1900년 일부)
```json
{
  "1900": [
    {"name": "소한", "month": 1, "day": 6, "time": "03:04"},
    {"name": "대한", "month": 1, "day": 20, "time": "20:33"},
    {"name": "입춘", "month": 2, "day": 4, "time": "14:52"},
    ...
  ]
}
```

### 24절기 목록 (배열 순서, 양력 연도 기준)
소한, 대한, 입춘, 우수, 경칩, 춘분, 청명, 곡우, 입하, 소만, 망종, 하지,
소서, 대서, 입추, 처서, 백로, 추분, 한로, 상강, 입동, 소설, 대설, 동지

### 사주 계산에서의 사용
- **연주(年柱) 판정**: 입춘 시각 기준. 출생이 입춘 이전이면 전년도 간지 사용.
- **월주(月柱) 판정**: 12절(입춘·경칩·청명·입하·망종·소서·입추·백로·한로·입동·대설·소한)
  기준. 해당 절의 절입 시각 이후부터 다음 절까지가 동일 월지.
- 중기(우수·춘분·곡우·소만·하지·대서·처서·추분·상강·소설·동지·대한)는
  월주 계산에 직접 사용하지 않으나 참고용.

### 시각(time) 필드
- KST(한국표준시, UTC+9) 기준
- `HH:MM` 24시간제
- 전 연도 전 절기 100% 채워짐 (검증 완료)

### 커버리지
- 149개 연도 × 24절기 = 3,576개 (누락 0건 검증 완료)

## 알려진 이슈

### 동지 절기 날짜 오차
- 1900~1902년: 사용자가 보유한 전통 만세력 이미지 기준으로 수기 수정 완료
- 1903년 이후: 미수정. KASI 천문계산값 기준 그대로.
- 전통 만세력과 현대 천문계산의 방법론 차이로 간혹 1일 오차 발생.
- 수정 계획: 만세력 이미지 재접근 가능 시 전수 대조.
