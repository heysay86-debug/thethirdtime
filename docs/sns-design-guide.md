## 4. 캐러셀 디자인 규격

### 기본 사양
- 크기: 1080 x 1350px (4:5)
- 장수: 6~9장
- 도구: Canva
- 폴더: `제3의시간 > 인스타그램 캐러셀`

### 레이아웃 구조 (장별)

```
┌─────────────────────┐
│                     │
│  ┌──────────┐       │  ← 말풍선 (첫 번째 카피)
│  │ 카피 1   │       │
│  └──┬───────┘       │
│     │               │
│   [복길 캐릭터]      │  ← 캐릭터 (표정 교체)
│                     │
│  ┌──────────┐       │  ← 말풍선 (두 번째 카피, 선택)
│  │ 카피 2   │       │
│  └──────────┘       │
│                     │
│ ┌─────────────────┐ │
│ │ 복길              │ │  ← 게임 대화창 (마지막 카피)
│ │ "핵심 한마디"      │ │
│ │              ▼   │ │
│ └─────────────────┘ │
└─────────────────────┘
```

### 구성 요소
- **배경**: 게임 스크린샷 (opacity 30~50%) 또는 단색 팔레트
- **상단**: 캐릭터 + 말풍선 1~2개 (시선 끌기)
- **하단**: 게임 대화창 UI (핵심 카피)
- **캐릭터**: 기본 복길 PNG + 장별 표정 교체
- **대화창**: 픽셀 보더 스타일 (서비스 내 DialogueBox와 동일)

### 장별 역할 (서비스 소개 캐러셀 기준)

| 장 | 역할 | 예시 |
|---|---|---|
| 1장 | 표지/훅 | "RPG형 사주풀이, 제3의시간" |
| 2장 | 자기소개 | 복길 등장, 캐릭터 인사 |
| 3장 | 몰입 | 세계관, 운명적 만남 |
| 4장 | 차별화 | 경쟁사와 다른 점 |
| 5장 | 공감 | 유저의 니즈 자극 |
| 6~7장 | 서비스 설명 | 사주/육효점 기능 소개 |
| 8장 | 안심/권유 | 어렵지 않다, 함께 하자 |
| 마지막 | CTA | 링크 + "만나러 오게" |

### 표정 가이드

| 감정 | 쓰이는 장면 |
|---|---|
| 기본/정면 | 표지, CTA |
| 자신감/웃음 | 자기소개, 전문성 강조 |
| 신비로운/미소 | 세계관, 육효점 |
| 진지한 | 차별화, 정통성 강조 |
| 따뜻한/미소 | 안심, 권유 |

## 7. 카피 톤앤매너

### 복길의 말투
- 해라체 + 고풍스러운 표현 ("~하겠네", "~일세", "~가보자고")
- 친근하지만 신비로운 느낌
- 유저를 "자네"로 호칭

### 금지
- 길흉 단정 ("좋은 사주", "나쁜 운")
- 특정 학자명/서적명 (이석영, 사주첩경)
- "운명은 정해져 있다" 류 표현
- "회원가입", "로그인" 용어

### 권장
- 비유 활용 (나무, 물, 불 등 오행 이미지)
- 자기이해 관점 ("알면 대비할 수 있다")
- RPG/게임 용어 자연스럽게 활용

## 10. Canva 작업 팁

- 1장을 완성한 뒤 **페이지 복제**로 나머지 장 생성
- 캐릭터 PNG는 같은 위치, **표정 PNG만 교체**
- 대화창 요소는 그룹화해서 텍스트만 수정
- 내보내기: PNG, 각 장 개별 저장
