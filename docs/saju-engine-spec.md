# 사주 계산 엔진 명세

사주팔자(연주·월주·일주·시주) 산출 로직. 이석영『사주첩경』 이론 기준.

---

## 입력

| 필드 | 타입 | 필수 | 설명 |
|---|---|---|---|
| `birth_date` | str `"YYYY-MM-DD"` | ✓ | 양력 생년월일 |
| `birth_time` | str `"HH:MM"` (24h) | 옵션 | 출생 시각. 미상 시 시주 제외 |
| `calendar` | `"solar"` \| `"lunar"` | ✓ | 입력 달력 유형 |
| `is_leap_month` | bool | 음력 시 필수 | 음력 윤달 여부 |
| `gender` | `"M"` \| `"F"` | ✓ | 대운 방향 결정용 |
| `timezone` | str | 옵션 | 기본 KST. 해외 출생 시 필요 |

## 출력

```json
{
  "year_pillar":  {"gan": "甲", "ji": "子"},
  "month_pillar": {"gan": "丙", "ji": "寅"},
  "day_pillar":   {"gan": "戊", "ji": "辰"},
  "hour_pillar":  {"gan": "壬", "ji": "子", "known": true},
  "birth": {
    "solar": "1990-02-15",
    "lunar": "1990-01-20",
    "time":  "14:30",
    "kst":   "1990-02-15T14:30:00+09:00"
  },
  "notes": ["..."]
}
```

---

## 계산 단계

### 0단계: 입력 정규화

1. `calendar == "lunar"`이면 `manse_data.json`(원본)에서 양력 변환.
   - 음력 윤달 처리: `is_leap: true`인 레코드 매칭.
2. 타임존 변환: 모든 계산은 KST 기준. 해외 출생 시 KST로 변환 후 진행.
3. 입력 검증: 범위 1900-01-01 ~ 2049-12-31 (lookup 테이블 범위).

### 1단계: 연주(年柱) 결정

**기준**: 입춘(立春) 절입 시각.

```
절입_시각 = jeolip[출생연도]에서 name=="입춘"인 항목의 (month, day, time)
if (출생_일시 < 입춘_시각):
    연주_기준연도 = 출생연도 - 1
else:
    연주_기준연도 = 출생연도
```

**간지 계산**: 기준연도에서 간지 추출.
- 60갑자 순환. 기준점: 1900년 = 경자(庚子)년.
- `천간 = "갑을병정무기경신임계"[(연도 - 4) % 10]`
- `지지 = "자축인묘진사오미신유술해"[(연도 - 4) % 12]`
  (서기 4년 = 갑자년 기준. 검증 필요)

### 2단계: 월주(月柱) 결정

**기준**: 12절(節). 중기(中氣)는 사용하지 않음.

12절 순서 (입춘부터):
| 절기 | 월지 | 해당 월 |
|---|---|---|
| 입춘 | 寅 | 1월 (정월) |
| 경칩 | 卯 | 2월 |
| 청명 | 辰 | 3월 |
| 입하 | 巳 | 4월 |
| 망종 | 午 | 5월 |
| 소서 | 未 | 6월 |
| 입추 | 申 | 7월 |
| 백로 | 酉 | 8월 |
| 한로 | 戌 | 9월 |
| 입동 | 亥 | 10월 |
| 대설 | 子 | 11월 |
| 소한 | 丑 | 12월 |

**월지 결정 알고리즘**:
```
출생_일시의 전후 12절 절입 시각을 jeolip에서 조회
출생_일시가 어느 절입~다음절입 구간에 속하는지로 월지 확정
경계 케이스: 연도를 넘어가는 소한(12월)의 경우 전년도 jeolip에서 조회
```

**월간(月干) 결정**: 연간에서 도출 (오호둔 월건법).
```
연간(天干)     | 정월(寅月)의 월간
갑·기         | 丙
을·경         | 戊
병·신         | 庚
정·임         | 壬
무·계         | 甲
```
월간은 정월(寅月) 기준에서 60갑자 순서대로 순행.

### 3단계: 일주(日柱) 결정

**기준**: `iljin.json` 단순 lookup.

```
일주 = iljin[출생_양력_날짜]
```

**자시(子時) 경계 처리**: 사주학의 야자시(夜子時) 논쟁.

- **조자시·야자시 분리설** (이석영 『사주첩경』 기준)
  - 23:00~23:59: 당일 자시(야자시)로 취급, **일주는 당일 유지**
  - 00:00~00:59: 익일 자시(조자시), **일주는 익일로 변경**
- **단일 자시설** (대안)
  - 23:00~00:59 전체를 익일 자시로 간주
  - 본 프로젝트는 **이석영 기준 = 분리설** 채택

기본 규칙:
```
if time in [00:00, 01:00):  일주 = iljin[출생_날짜]
elif time in [23:00, 24:00): 일주 = iljin[출생_날짜] (당일 유지)
else:                        일주 = iljin[출생_날짜]
```
(실질적으로 일주는 출생 양력 날짜 기준 고정. 시주 계산에서만 자시 분기)

### 4단계: 시주(時柱) 결정

**시각 미상**: 시주 제외, `known: false` 표시. 계산 불가함을 명시.

**시지(時支) 결정**: M8에서 경도 보정된 진태양시를 입력받아 정시제 기준으로 판단.
※ KST 직접 입력 금지. 반드시 진태양시 변환 후 전달할 것 (M8 책임).

| 시진 | 진태양시 |
|---|---|
| 子 (조자시) | 00:00 ~ 02:00 |
| 丑 | 02:00 ~ 04:00 |
| 寅 | 04:00 ~ 06:00 |
| 卯 | 06:00 ~ 08:00 |
| 辰 | 08:00 ~ 10:00 |
| 巳 | 10:00 ~ 12:00 |
| 午 | 12:00 ~ 14:00 |
| 未 | 14:00 ~ 16:00 |
| 申 | 16:00 ~ 18:00 |
| 酉 | 18:00 ~ 20:00 |
| 戌 | 20:00 ~ 22:00 |
| 亥 | 22:00 ~ 24:00 |
| 子 (야자시) | 24:00 ~ 26:00 (= 다음날 00:00~02:00) |

**시간(時干) 결정**: 일간에서 도출 (오서둔 시건법).
```
일간(天干)     | 자시의 시간
갑·기         | 甲
을·경         | 丙
병·신         | 戊
정·임         | 庚
무·계         | 壬
```
시간은 자시 기준에서 60갑자 순서대로 순행.

**야자시 특례**: 진태양시 기준 23:00~24:00 출생 시 (= 다음날 -1~0시)
- 일주: 당일 유지
- 시주: **익일 일간 기준**으로 시간 계산

## 경계 케이스 체크리스트

- [ ] 입춘 당일 출생 (시각 비교 필수)
- [ ] 12절 당일 출생 (월주 경계)
- [ ] 자정 전후 출생 (일주 경계)
- [ ] 23:00~23:59 출생 (야자시)
- [ ] 음력 윤달 출생
- [ ] 시각 미상
- [ ] 해외 출생 (타임존)
- [ ] 1900년 1월 / 2049년 12월 (lookup 경계)
- [ ] 윤년 2/29
- [ ] 서머타임(DST) 시기 출생 (한국은 1948~1951, 1955~1960, 1987~1988 시행)
  → 이 프로젝트는 **KST 표준시 고정**. 서머타임 적용 여부는 사용자 입력값에 따름.

## 검증 전략

**레퍼런스 사주 데이터셋**: 만세력으로 검증 가능한 유명인 생년월일 또는
공개 사주 예제를 10건 이상 수집하여 단위 테스트로 자동 검증.

**검증 항목**:
1. 간지 값 정확성 (연·월·일·시주 각각)
2. 경계 케이스 처리 (위 체크리스트)
3. 음력 입력의 양력 변환 정확성
