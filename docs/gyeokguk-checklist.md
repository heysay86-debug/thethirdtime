# 격국(格局) 판정 체크리스트

최종 업데이트: 2026-04-15
이론 기준: 이석영 『사주첩경』

본 문서는 `gyeok_guk.ts` 구현과 격국 결과 출력 전 검증 절차의 기준이 된다.
격국 판정은 명리학에서 가장 예외가 많은 영역이므로 **최종 출력 전 반드시 본 체크리스트를 통과**해야 한다.

---

## 1. 판정 순서 (Top-Down)

```
[1단계] 외격 성립 여부 확인 (재관식 유무 체크)
    ↓ 외격 성립 시 → 외격 세부 분류
    ↓ 미성립 시
[2단계] 특수 내격 확인 (건록격·양인격)
    ↓ 해당 시 → 특수 내격 처리
    ↓ 해당 없음
[3단계] 일반 내격 판정 (월지 지장간 투출 기준)
    ↓
[4단계] 파격·합거·충격 검증
    ↓
[5단계] 최종 출력 전 검증 리스트 통과
```

---

## 2. 1단계 — 외격 성립 여부

### 종격(從格) 성립 조건 (엄격)

**공통 조건**: 일간이 의지할 비겁·인성이 극히 미약하거나 없어야 함.

| 종격 유형 | 종하는 세력 | 성립 조건 | 성립 예시 |
|---|---|---|---|
| 종강격(從强格) | 비겁+인성 일색 | 재·관·식이 없거나 완전히 무력 | 壬水 일간에 金水 가득 |
| 종왕격(從旺格) | 비겁 일색 | 인성도 거의 없고 비겁만 가득 | 甲木 일간에 寅卯 가득 |
| 종아격(從兒格) | 식상 일색 | 식상 가득, 인성 無 | 甲木 일간에 巳午 가득 |
| 종재격(從財格) | 재성 일색 | 재성 가득, 비겁 無 | 甲木 일간에 辰戌丑未 가득 |
| 종살격(從殺格) | 관살 일색 | 관살 가득, 인성·비겁 無 | 甲木 일간에 申酉 가득 |

**중요 검증**: 재관식 중 하나라도 세력이 있으면 종격 불성립. 점수만으로 종격을 판정하지 말 것.

### 진종(眞從) vs 가종(假從)

- **진종**: 일간을 도울 글자가 완전히 없음. 안정적 종격.
- **가종**: 미약한 비겁·인성이 있으나 무력하여 종함. 운에서 비겁·인성이 강해지면 반란(흉).

### 화격(化格) 성립 조건

일간이 다른 천간과 합하여 오행이 변하는 격.

| 합화 | 변화 오행 |
|---|---|
| 甲己合 | 土 |
| 乙庚合 | 金 |
| 丙辛合 | 水 |
| 丁壬合 | 木 |
| 戊癸合 | 火 |

**성립 엄격 조건**:
1. 일간이 합하는 상대 천간이 월간 또는 시간에 있어야 함 (근접 위치)
2. 합하여 화한 오행이 **월령을 얻어야 함**
3. 사주 전체가 화한 오행의 세력으로 가득해야 함
4. 화한 오행을 극하는 글자가 없어야 함

**미성립 시**: 단순 합으로만 처리. 화격 판정 금지.

## 3. 2단계 — 특수 내격 (건록격·양인격)

### 건록격(建祿格)

월지가 일간의 건록인 경우. 비견(比肩)이 월지에 위치.

| 일간 | 건록 (월지) |
|---|---|
| 甲 | 寅 |
| 乙 | 卯 |
| 丙·戊 | 巳 |
| 丁·己 | 午 |
| 庚 | 申 |
| 辛 | 酉 |
| 壬 | 亥 |
| 癸 | 子 |

특징: 일반 내격과 달리 월지 지장간 투출로 격을 뽑지 않음. 재·관을 용신으로 쓰는 방향.

### 양인격(羊刃格)

월지가 일간의 양인인 경우. 겁재(劫財)가 월지에 위치하며 매우 강한 비겁.

| 일간 | 양인 (월지) |
|---|---|
| 甲 | 卯 |
| 丙·戊 | 午 |
| 庚 | 酉 |
| 壬 | 子 |

※ 음간(乙·丁·己·辛·癸)은 양인 없음 (학파에 따라 인정하는 경우 있음)

특징: 살인상정(煞刃相停) 구조가 핵심. 관살을 용신으로 제압.

## 4. 3단계 — 일반 내격 판정

### 투출 확인 범위

**천간 전체 검사**: 연간, 월간, 시간 모두 확인 (일간은 본인이므로 제외)

```
월지 지장간 → 연간/월간/시간 중 동일 글자 있음? → 투출
```

### 투출 우선순위

1. 월간에 투출 → 최우선 격
2. 시간 또는 연간에 투출 → 차순위 격
3. 복수 투출 시: 본기(本氣) 투출 > 중기 > 여기

### 투출 없음 시

월지 본기(本氣) 기준으로 격을 결정.

### 10정격(十正格) 분류

| 격 이름 | 조건 (일간 기준) |
|---|---|
| 정관격 | 월지 또는 투출 천간이 정관 |
| 편관격(칠살격) | 편관 |
| 정재격 | 정재 |
| 편재격 | 편재 |
| 정인격 | 정인 |
| 편인격 | 편인 |
| 식신격 | 식신 |
| 상관격 | 상관 |
| 건록격 | (2단계 참조) |
| 양인격 | (2단계 참조) |

※ 비견격은 일반적으로 건록격·비견격으로 흡수됨

## 5. 4단계 — 파격·합거·충격 검증

격이 잠정 결정된 후 아래 검증을 거친다. 조건 해당 시 격이 손상되거나 변경된다.

### 합거(合去) 체크

- 격을 이루는 글자(월간 또는 투출 글자)가 다른 천간과 합으로 묶이면 격이 약화됨
- 예: 丁壬合 — 월간 丁(정재)이 일간 壬과 합 → 정재의 격으로서의 역할 제한
- 처리: 격명은 유지하되 "합거로 약화" 플래그 추가, 또는 다른 격으로 전환

### 충격(冲格) 체크

- 월지가 사주 내 다른 지지에게 충을 당하면 격이 파괴됨
- 예: 子월인데 사주에 午가 있으면 子午충 → 격 손상
- 처리: "충으로 파격" 플래그, 변통 필요

### 파격(破格) 조건

| 격 | 파격 조건 |
|---|---|
| 정관격 | 상관이 정관을 극 (상관견관) |
| 편관격(칠살) | 제화(制化) 없이 강하기만 함 |
| 정인격·편인격 | 재성이 인성을 극 (재극인) |
| 정재격·편재격 | 비겁이 재성을 극 (비겁쟁재) |
| 식신격 | 편인이 식신을 극 (편인도식) |
| 상관격 | 정관을 봄 (상관견관) |

### 형(刑)·해(害) 영향

- 월지와 다른 지지 간 형·해 관계가 있으면 격이 부분 손상
- 이 사주 예: 酉월 + 戌일 = 酉戌해 → 정인·편인격의 인성 기능이 일부 손상

## 6. 5단계 — 최종 출력 전 검증 리스트

격국 결과 출력 전 반드시 아래 항목을 모두 통과해야 한다.

### 6-1. 판정 근거 검증

- [ ] 1단계(외격) 판정 시 재·관·식 세력 유무를 명시적으로 확인했는가?
- [ ] 종격 판정 시 점수만이 아니라 재·관·식 **부재**를 확인했는가?
- [ ] 화격 판정 시 4가지 엄격 조건을 모두 통과했는가?
- [ ] 2단계(건록·양인) 해당 여부를 확인했는가?
- [ ] 3단계(일반 내격) 투출 확인 시 연간·월간·시간을 **모두** 검사했는가?
- [ ] 투출 글자가 2개 이상일 때 우선순위를 적용했는가?

### 6-2. 파격 검증

- [ ] 격을 이루는 글자의 합거 여부를 확인했는가?
- [ ] 월지의 충 여부를 확인했는가?
- [ ] 격별 고유 파격 조건(상관견관, 재극인 등)을 확인했는가?
- [ ] 형·해로 인한 손상을 확인했는가?

### 6-3. 일관성 검증

- [ ] 신강/신약 판단 결과와 격국이 모순되지 않는가?
  예: 종강격인데 재관이 강함 → 모순
- [ ] 격국과 일간 오행의 관계가 명리학적으로 성립하는가?
- [ ] 시주(시간·시지)를 계산 전체에 반영했는가?

### 6-4. 출력 구조 검증

- [ ] 격국명 외에 성격/파격 여부를 명시했는가?
- [ ] 파격인 경우 그 원인(합거·충·재극인 등)을 명시했는가?
- [ ] 특수 사례(외격·화격·건록·양인) 해당 여부를 구분 표시했는가?
- [ ] 시주 미상 시 잠정 판정임을 표시했는가?

## 7. 구현 가이드 (gyeok_guk.ts)

### 함수 구조 권장안

```typescript
interface GyeokGukResult {
  name: string;              // 격 이름 (예: "편인격")
  category: '내격' | '외격' | '특수'; // 분류
  subtype?: string;          // 세부 분류 (예: 종강격의 경우)
  state: '성격' | '파격' | '약화'; // 상태
  破格_causes?: string[];    // 파격 원인 (있을 시)
  basis: {                   // 판정 근거
    method: '투출' | '본기' | '월지' | '합화';
    source_char?: string;    // 격을 결정한 글자
    source_position?: string; // 위치 (연간/월간/시간 등)
  };
  warnings?: string[];       // 경고·주의 사항
  confidence: 'high' | 'medium' | 'low'; // 확신도
}

function determineGyeokGuk(saju: SajuData): GyeokGukResult {
  // 1단계: 외격 검사
  const ogyeok = checkOgyeok(saju);
  if (ogyeok.isEstablished) return buildOgyeokResult(ogyeok);

  // 2단계: 특수 내격
  const special = checkSpecialNaegyeok(saju);
  if (special) return special;

  // 3단계: 일반 내격
  const ilban = determineIlbanNaegyeok(saju);

  // 4단계: 파격·합거·충격 검증
  const verified = verifyGyeokGuk(ilban, saju);

  // 5단계: 최종 검증 리스트 통과 확인
  const finalResult = runFinalChecks(verified, saju);

  return finalResult;
}
```

### 각 단계 분리 원칙

- 1~5단계는 **각각 독립 함수**로 분리
- 단계별 단위 테스트 필수
- 각 단계 결과를 로그로 남겨 추적 가능하도록 구현

## 8. 레퍼런스 사주 카테고리 (테스트 필수)

격국 로직 검증을 위해 아래 각 카테고리당 최소 1건 이상의 레퍼런스 사주가 필요하다.

- [ ] 정관격 성격 / 파격(상관견관)
- [ ] 편관격 제화 / 미제화
- [ ] 정인격 성격 / 파격(재극인)
- [ ] 편인격 성격 / 파격
- [ ] 정재격·편재격 성격 / 파격(비겁쟁재)
- [ ] 식신격 성격 / 파격(편인도식)
- [ ] 상관격 성격 / 파격(상관견관)
- [ ] 건록격
- [ ] 양인격
- [ ] 종강격 / 종왕격 / 종아격 / 종재격 / 종살격 (각각)
- [ ] 화격 성립 / 불성립
- [ ] 투출 글자 합거 케이스
- [ ] 월지 충 케이스
- [ ] 가종격 (운에 따른 반란 케이스)

## 9. 기존 레퍼런스 사주의 재검토 필요성

현재 레퍼런스에 설정된 판정 중 재검증이 필요한 것:

- **1986-09-15 01:17 壬戌일 丁酉월**
  - 현재: "정인격 (재극인 파격)"
  - 재검토 포인트:
    - 시간 庚의 투출 인정 시 **편인격**으로 변경 가능
    - 壬丁合 합거 처리 필요
    - 종강격 오판 방지 (재관식 엄격 확인)
  - 권장 판정: **편인격(성격 또는 합거 약화)**, 종강격 아님

## 10. 참고 원전·해석 유파

격국 판단에서 학파별 견해가 갈리는 부분은 `docs/interpretation-policy.md`의 5유파 보강 원칙을 따른다.

- **1차 기준**: 이석영 『사주첩경』
- **2차 보강** (필요 시):
  - 자평진전: 격국 성패 판단 체계
  - 적천수: 일간 기세 중심 해석
  - 명리정종: 십성 배치
  - 궁통보감: 조후 관점

본 체크리스트는 1차 기준(이석영)에 따른 엄격 판정 절차이며, 2차 유파 관점은 해석 출력 단계(LLM 게이트웨이)에서 병기한다.
